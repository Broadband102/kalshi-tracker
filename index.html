📖 How This Works
1️⃣ What is Kalshi?
Kalshi is a legal prediction market exchange. You bet YES or NO on real-world events — like "Will the Fed raise rates?" or "Will it snow in NYC this week?" Each contract pays $1 if correct, $0 if wrong. You buy contracts for less than $1, aiming to sell higher or hold to resolution.
2️⃣ The 10-Play Strategy
Split your bankroll across 10 different trades (~$10 each). This way no single loss kills you. If 6 win at +5% and 4 lose at -3%, your total is: +$3.00 - $1.20 = +$1.80 profit (1.8% on $100). That's the engine! Aim for 2% per day across the whole portfolio.
3️⃣ The Tax Reality
Kalshi winnings are taxed as ordinary income (not capital gains). At 22%, to net $100 profit from your $100 starting bankroll, you need to make ~$128 gross profit. So your true 2× target is ~$228 gross. The projector already accounts for this!
4️⃣ Daily Routine (5 min)
☀️ Morning: Open this app → Daily Brief → Generate → Check 10 Plays
📊 Enter trades: Go to Portfolio tab → Add positions you actually placed
🔄 Update prices: Edit current price field as markets move
📥 Backup: Download CSV whenever you close the browser!
5️⃣ The Flip — Don't Hold to Zero!
You don't have to wait for the event to resolve! If you buy YES at 60¢ and news moves the market to 70¢, sell for +16.7% profit and redeploy immediately. This is the fastest way to compound. Most beginners hold too long — lock in the price move!
6️⃣ Hard Rules 🛑
🔴 Never risk more than 15% on one trade
🔴 If a position drops 30%, exit. No averaging down.
🔴 If you lose 15% of bankroll in one day, stop trading for 24 hours
🔴 Never chase a play after it already moved
🔴 This is $100 fun money — don't stress about it!

⚙️ Getting the AI Working (One-Time Setup)

The Anthropic API blocks direct browser calls for security (CORS policy). You need a tiny free "relay" that sits between your browser and the API. Takes about 5 minutes using Cloudflare's free plan.

1️⃣ Create a Free Cloudflare Account
Go to dash.cloudflare.com/sign-up and sign up free.
No credit card needed. Cloudflare Workers free tier = 100,000 requests/day.
2️⃣ Create a Worker
In Cloudflare dashboard → Workers & Pages → Create → Create Worker
Name it something like kalshi-proxy → click Deploy
3️⃣ Paste This Code Into the Worker
Click Edit Code, delete everything, paste this:
export default { async fetch(request, env) { if (request.method === 'OPTIONS') { return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'POST, OPTIONS', 'Access-Control-Allow-Headers': 'Content-Type', }}); } const body = await request.text(); const resp = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'anthropic-version': '2023-06-01', 'x-api-key': env.ANTHROPIC_API_KEY, }, body, }); const data = await resp.text(); return new Response(data, { headers: { 'Content-Type': 'application/json', 'Access-Control-Allow-Origin': '*', }}); } };
4️⃣ Add Your Anthropic API Key as a Secret
In the Worker page → Settings → Variables and Secrets
→ Add Secret → Name: ANTHROPIC_API_KEY
→ Value: your key from console.anthropic.com/keys
→ Click Deploy again to save.
