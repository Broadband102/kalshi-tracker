📖 How This Works
1️⃣ What is Kalshi?
Kalshi is a legal prediction market. Bet YES or NO on real events — Fed rates, weather, politics. Each contract pays $1 if right, $0 if wrong. You buy contracts cheap and sell high (or hold to resolution).
2️⃣ The 10-Play Strategy
Split $100 across 10 trades (~$10 each). If 6 win +5% and 4 lose -3%: +$3.00 - $1.20 = +$1.80 profit. That's 1.8% on your bankroll! Aim for 2%/day across the whole portfolio.
3️⃣ Tax Reality
Kalshi winnings = ordinary income tax. At 22%, to net $100 profit you need ~$128 gross. True 2× target = ~$228 gross. Projector shows both lines!
4️⃣ The Flip — Don't Hold to Zero!
Buy YES at 60¢, news moves market to 70¢ → sell for +16.7% profit and redeploy. Lock in the price move, don't wait for resolution!
5️⃣ Hard Rules 🛑
🔴 Max 15% on one trade   🔴 Exit if down 30%   🔴 Stop trading if down 15% in one day   🔴 This is $100 fun money — don't stress!
⚙️ One-Time Proxy Setup (5 min, free)

Browsers block direct calls to the Anthropic API (CORS security). You need a free Cloudflare Worker relay. Do this once and the AI buttons work forever.

1️⃣ Sign up free at Cloudflare
Go to dash.cloudflare.com/sign-up — no credit card needed.
2️⃣ Create a Worker
Dashboard → Workers & Pages → Create → Start with Hello World!
Name it kalshi-proxy → click Deploy
3️⃣ Replace the code
Click Edit Code → select all → delete → paste this:
export default { async fetch(request, env) { if (request.method === 'OPTIONS') { return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'POST, OPTIONS', 'Access-Control-Allow-Headers': 'Content-Type', }}); } const requestBody = await request.clone().text(); const resp = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'anthropic-version': '2023-06-01', 'x-api-key': env.ANTHROPIC_API_KEY, }, body: requestBody, }); const data = await resp.text(); return new Response(data, { status: resp.status, headers: { 'Content-Type': 'application/json', 'Access-Control-Allow-Origin': '*', }}); } };
4️⃣ Add your Anthropic API key
Worker page → Settings → Variables and Secrets → Add Secret
Name: ANTHROPIC_API_KEY
Value: your key from console.anthropic.com/keys
→ Click Deploy again
